# Spotify App

For demo click [HERE](https://antonskliarov.github.io/spotify-app/)

## Tech stack

1. HTML5
1. SASS
1. TypeScript
1. React (v.17.0.1)

## Install and Usage
*****
  * Fork and clone this repository
  * Run `npm install` in your terminal
  * Run `npm start`
  * Open `http://localhost:3000/` at your browser
